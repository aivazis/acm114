Course outline

Here is an outline of the material that will get covered in roughly chronological order

  • Scientific computing preliminaries
    • The role of computation in science
    • Numerical methods: matrices, integrating ODE/PDE, stochastic processes, optimization
    • Motivation for parallel programming: resolution
    • Hardware and software evolution
    • Software engineering survival skills
    • Algorithms: specification, design, analysis
    • Basic data structures
    • Advanced data structures
  • Overview of concurrent algorithms and their operating environments
    • PREM: an abstract parallel machine
    • Memory access: concurrent vs. exclusive
    • Memory models: shared vs. distributed memory
    • Process synchronization: explicit vs. implicit
    • Performance, parallel efficiency and scalability
    • Static and dynamic algorithms
  • HPC: high-performance scientific computing
    • Resource planning
    • Staging and launching a parallel calculation
    • Execution environments
    • Monitoring
    • Data harvesting
    • Post-processing
    • Visualization and knowledge mining
  • Embarrassingly parallel problems
    • Random number generators
    • Monte Carlo integration
      • Introduction
      • Integrating using a fixed number of samples
      • Achieving a desired accuracy
    • Optimization
      • Introduction
      • Parallel optimization strategies
  • Structured grids
    • The Laplace equation
    • Two and three dimensional arrays
    • Domain decomposition strategies
    • Building the communication maps
    • Synchronization and information exchange
    • Adaptive mesh refinement
  • Unstructured grids
    • Meshes as graphs
    • Decomposition by graph partitioning
    • Building the communication maps
    • Synchronization and information exchange
    • Communicating topological changes
    • Load balancing
  • Non-local problems
    • The gravitational n-body problem in two dimensions
    • Spatial data structures
  • Dense and sparse matrices
  • Advanced application design
    • Designing for long software life-cycles
    • Mixed language programming
    • Interpreted languages and high performance computing
    • Components and component framework
