Schedule of lectures

This page contains outlines of the lectures given during the Winter 2012 term. You can use the calendar on the left to navigate to a specific lecture where you will also find instructions on how to download the lecture notes in PDF form. If you are interested in the LaTeX source code, you will have to gain access to the bazaar source repository. Let me know and I will send you instructions.

Please note that there are no lectures on January 16 and February 20; they are Institute holidays. The last day of class is March 9, and final projects are due on March 16.

Wednesday, January 4

Introductions. The syllabus. Course scope and outline. Motivation and overview of high performance computing in science. Informal student survey: background, experience, personal computing platforms, access to computational resources.

Friday, January 6

Overview of algorithms and their analysis. Insertion sort as an example of an incremental algorithm, merge sort as an example of a divide and conquer algorithm. Pseudocode implementations. Asymptotic notation for the running times of algorithms.

Monday, January 9

Software engineering survival skills. Quick introduction to ssh, private and public keys, identity management using ssh-agent. The bzr source control system: creating repositories; pushing and pulling from remote servers; adding and committing content. Using the class homework repository as an example. Workflows.

Wednesday, January 11

Overview of parallel systems. Taxonomy, architectural issues and tradeoffs. Categories of parallel architectures and their memory hierarchies. Overview of parallel programming paradigms. Classification and programming models for parallel algorithms. Effects of communication. Factors determining performance. Abstract computational models.

Friday, January 13

Illustration of parallel programming models with a simple problem: a global reduction. Shared and distributed address spaces; threads and message passing. Example pitfalls: race conditions. Layout of a SIMD machine and data parallel programming. Clusters of SMPs and hybrid programming models. Bulk synchronous programming and algorithmic phases.
